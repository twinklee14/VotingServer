A secure RESTful backend application for an online voting system built using Node.js, Express.js, and MongoDB. The system supports user authentication, role-based access control, candidate management, and secure voting.

Features:
JWT-based Authentication for secure access
Role-based Authorization (Admin & Voter)

Admin Functionalities:
Add, update, and delete candidates

Voting System:
Users can vote only once
Admins are restricted from voting
Vote Count API with results sorted in descending order

Tech Stack:
Node.js
Express.js
MongoDB
Mongoose
JWT (JSON Web Tokens)
dotenv
